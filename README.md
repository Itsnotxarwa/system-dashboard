# Admin Dashboard

A modern Admin Dashboard built with React, Tailwind CSS, and Lucide React icons. The dashboard displays key metrics (KPIs), sessions, tenants, configuration status, and analytics in a clean, responsive interface.

## Installation

1. Clone the repository:

git clone https://github.com/Itsnotxarwa/admin-dashboard.git
cd admin-dashboard

2. Install dependencies:

npm install 

3. Run the development server:

npm run dev

4. Open your browser at: 

http://localhost:5173/admin-dashboard
